# 一、基本语法

## 1. 完整语法格式

```
[capture-list](parameters) mutable -> return-type {
    // 函数体
}
```

## 2. 最简单的 Lambda 表达式

```
[]{};  // 一个不捕获任何变量、不接收参数、不做任何事情的Lambda
```



# 二、组成部分

## 1. 捕获列表 (capture-list)

- **作用**：指定Lambda可以访问的外部变量
- 捕获方式：
  - `[]`：不捕获任何变量
  - `[=]`：以值方式捕获所有外部变量
  - `[&]`：以引用方式捕获所有外部变量
  - `[x, &y]`：混合捕获（x值捕获，y引用捕获）
  - `[this]`：捕获当前类的this指针



## 2. 参数列表 (parameters)

- 与普通函数的参数列表类似
- 可以省略参数列表（当不需要参数时）



## 3. mutable修饰符

- 允许修改值捕获的变量（默认情况下值捕获的变量是const的）
- 不影响引用捕获的变量



## 4. 返回类型 (return-type)

- 可以显式指定返回类型
- 可以省略（编译器会自动推导返回类型）



## 5. 函数体

- 要执行的代码
